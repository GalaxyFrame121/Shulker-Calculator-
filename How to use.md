EN:

-----HOW TO USE THE SC-----

1.In the top you have 4 buttons,
   you have the Language part wich lets you change beetween Languages, to fully change the language you need to first choose the language, then reload the page
   you have the "Shulker/Chest" and "Large Chest" buttons, the second one leads you to another calculator made for large chests
   you have on/off like button, it toggles/disables dark mode
2. Down the header you can see a part where you can input the number of stacks of the item that you need to storage, put a integrer number, DO NOT USE DECIMALS
3. Finally you have the calculate button that will calculate how many Shulkers/Chests you need to storage all the stacks you put on the input
   It will also say how many slots will be empty of the last Shulker/Chest to fill to make it more specific

ES:

-----COMO USAR LA SC ---

1. Arriba del todo tendras 4 botones,
   tienes el boton de idioma que te deja cambiar entre idiomas, para cambiar el idioma por completo primero debes seleccionar el idioma y despues recargar la pagina
   tienes los botones de "Shulker/Chest" y "Large Chest", el segundo te llevara a otra calculadora hecha para cofres dobles
   tienes un boton parecido a uno de encender y apagar, al activarlo se activa el modo oscuro
2. Debajo del encabezado tienes un lugar donde escribir el numero de stacks del item que quieres almacenar, pon un numero entero, NO USES DECIMALES
3. Por ultimo tienes el boton de calcular que te dira cuantas Shulkers/Cofres necesitas para guardar todos los stacks que pongas en la parte de arriba
   Tamvien dira cuantos huecos vacios tendra la ultima shulker/cofre a llenar para hacerlo mas especifico
